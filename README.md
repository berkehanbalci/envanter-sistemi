# Envanter Sistemi
Satıcıların sahip olduğu ürünleri listelemeye olanak sağlayan envanter sistemi.

## Özellikler
Sistemde ürün, fiyat ve stok ekleme gibi özellikler mevcuttur. Aynı zamanda aynı üründen varsa sistem otomatik algılayıp varolan stoğa ekleme yapar.

## Kullanım
İlk adım editörünüzden `main.py` adlı dosyayı açmak. Daha sonra terminale `python main.py` yazarak uygulamayı çalıştırabilirsiniz.

## Kullanılan Teknolojiler

- Python
- SQLite
